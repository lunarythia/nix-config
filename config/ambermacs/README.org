* Instructions

Install [[https://github.com/plexus/chemacs2][Chemacs2]] to =~/.emacs.d=, then write the following to =~/.emacs-profiles=:
#+begin_src emacs-lisp :tangle no

  (("default" . ((user-emacs-directory . "/path/to/this/repo"))))

#+end_src

Open this file in Emacs and run run ~M-x org-babel-tangle~, then restart Emacs.  In the future, saving the file /should/ automatically tangle it.

If something goes wrong with installing packages, you may have to manually do ~M-x package-install RET [insert name of package]~.

In addition, you may need to install the package ~nerd-icons~ and run ~M-x nerd-icons-install-fonts~.

On macOS it is necessary to ~brew install coreutils~.

** Hydra

Hydra allows you to hit a key and then within a certain timeframe, hit other keys once or multiple times to quickly cycle through or toggle things. For example, after hitting ~SPC t s~, within 5 seconds, you can hit ~j~ and ~k~ to zoom in or out multiple times.

#+begin_src emacs-lisp

  (use-package hydra)

  (defhydra hydra-text-scale (:timeout 5)
    ("j" text-scale-increase "in")
    ("k" text-scale-decrease "out")
    ("0" (text-scale-set 0) "reset")
    ("f" nil "exit" :exit t))

  (defhydra hydra-pdf-scale (:timeout 5)
    ("j" pdf-view-enlarge "in")
    ("k" pdf-view-shrink "out")
    ("0" pdf-view-fit-page-to-window "reset")
    ("f" nil "exit" :exit t))

  (defhydra hydra-change-window (:timeout 5)
    ("j" evil-window-next "clockwise")
    ("k" evil-window-prev "counterclockwise")
    ("f" nil "exit" :exit t))
  (aw/leadkey
    "ww" '(hydra-change-window/body :which-key "change window"))

#+end_src
